### variable
- basic building block, which can store, access and modify the values
- const
    - `const lastName = 'Kundala'`
    - can't reassign or redeclared,
    - needs to be initialized while declaration.
- let
    - `let name = 'Aawni'`
    - can be reassigned and redeclare.
- var
    - `var value = 'Abc'`
